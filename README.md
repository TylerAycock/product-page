Functionality:

- Designed with optimal layout for the site depending on their device's screen size
- Hover states for all interactive elements on the page
- Interactive lightbox gallery by clicking on the large product image
- Image carousel
- Add items to the cart
- View the cart and remove items from it


## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px
